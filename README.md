# 📊 Excel Data Analytics Dashboards

This repository contains **interactive data analytics dashboards built using Microsoft Excel**.  
These dashboards analyze sales data to generate meaningful business insights using **Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques**.

The repository currently includes two dashboard projects:

* **Supermart Grocery Sales Dashboard**
* **Adidas Sales Dashboard**

Both dashboards transform raw sales data into **interactive visual insights** that help understand business performance and support data-driven decision making.

---

# 🛒 Supermart Grocery Sales Dashboard (Excel)

## 📌 Project Overview
An **interactive sales dashboard built using Microsoft Excel** to analyze grocery store sales data.  

The dashboard helps identify **sales trends, regional performance, product profitability, and discount patterns** across different product categories.  

Using **Pivot Tables, Pivot Charts, and Slicers**, users can explore the data and gain meaningful insights about business performance.

---

## 🎯 Business Questions Answered
1. Which category has the most sales?  
2. How much have sales increased year by year?  
3. How do sales compare across different regions?  
4. Which category gives the highest discount?  
5. Which category generates the highest profit?  
6. How is sales distributed across regions?  

---

## 📊 Key Performance Indicators (KPIs)
* **Total Sales (Overall Revenue):** 14,956,982  
* **Total Number of Orders:** 9,994  

---

## 📈 Dashboard Visualizations
- **Sales by Category**  
- **Sales by City & Region**  
- **Average Discount by Category**  
- **Yearly Sales Trend (2015–2018)**  
- **Sales Distribution by Region**  
- **Profit by Category**  

---

## 🎛 Interactive Features
- Month slicer filters for seasonal analysis  
- Interactive exploration of sales trends  

---

## 🛠 Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers  
- Data Visualization

# 👟 Adidas Sales Dashboard (Excel)

## 📌 Project Overview
An **interactive Adidas Sales Dashboard built using Microsoft Excel** to analyze sales performance across **cities, states, gender segments, and sales channels**.  

The dashboard highlights **top performing outlets, profitable cities, gender-based sales distribution, and monthly sales trends**.

---

## 🎯 Business Questions Answered
1. Which outlet generates the highest sales?  
2. Which city contributes the most revenue?  
3. How are sales distributed between men and women?  
4. Which states generate the most sales?  
5. What are the monthly sales trends?  
6. Which sales channel performs best (In-store, Online, Outlet)?  
7. Which sales method has the highest operating margin?  

---

## 📊 Key Performance Indicators (KPIs)
* **Top Outlet by Sales:** New York  
* **Top Store Sales:** Miami  
* **Top Store Profit:** Albany  
* **Top Online Sales:** Charleston  
* **Top Online Profit:** Charleston  
* **Top Outlet Profit:** New York  

---

## 📈 Dashboard Visualizations
- **Sales by Gender**  
- **Top 10 States by Sales**  
- **Monthly Sales Trends**  
- **Top 5 Cities by Sales**  
- **Average Operating Margin by Sales Method**  
- **Sales by Sales Method**  

---

## 🎛 Interactive Features
- Slicer filters for **Year** and **Gender**  
- Dynamic filtering for interactive analysis  

---
